# pblfetch
simple c++ fetch script with cute cactus <3
# to use
```shell
git clone https://github.com/itspabloo/pblfetch
```
```shell
cd pblfetch
```
```shell
mkdir -p ~/.config/pblfetch
```
```shell
cp art.txt ~/.config/pblfetch
```
```shell
g++ main.cpp -o main
```
```shell
sudo cp main /usr/local/bin/pblfetch
```
```shell
pblfetch
```
